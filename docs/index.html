Documentation

Ceci est la partie frontend du test technique visant à évaluer les compétences en développement d'interfaces React et en API Node.js, ainsi que la capacité à gérer efficacement un fichier CSV. Cette application est développée avec ReactJS et initialisée avec Vite pour assurer un chargement rapide pendant la phase de développement. Vous trouverez ci-dessous la documentation de chaque composant.

Allons-y!

Intégration du Projet : Jira et GitHub

Dans ce projet, j'ai intégré Jira avec GitHub pour assurer une collaboration fluide et une gestion de projet améliorée. Cette intégration sert plusieurs objectifs et offre de nombreux avantages :

  • Suivi Efficace des Problèmes : Gérez et priorisez de manière transparente les problèmes pour des efforts de développement ciblés.
  • Mises à Jour en Temps Réel : Restez informé avec des mises à jour automatiques sur la progression du projet et les changements.
  • Vue Holistique du Projet : Maintenez une perspective globale sur le cycle de vie du projet, de la planification au déploiement.
  • Rapports Avancés : Exploitez les analyses de Jira pour des informations approfondies, facilitant la prise de décision et l'analyse des performances.

La documentation des composants:

FileCard

Composant FileCard pour afficher des informations sur le fichier sélectionné.

FileCard
Parameters
props (Object) Les propriétés du composant.
Name Description
props.selectedFile Object L'objet de fichier sélectionné.
props.uploadProgress number Le pourcentage de progression du téléchargement.
props.uploadStatus string Le statut de téléchargement.
props.clearFileInput function Fonction pour effacer le fichier sélectionné.
Returns
JSX.Element: Composant React.
Example
// Exemple d'utilisation :
<FileCard
  selectedFile={selectedFile}
  uploadProgress={uploadProgress}
  uploadStatus={uploadStatus}
  clearFileInput={clearFileInput}
/>
